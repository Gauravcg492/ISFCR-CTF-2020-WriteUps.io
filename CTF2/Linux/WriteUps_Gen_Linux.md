Challenge 3
-----------

The flag is in the given text file. Can you find it?

Points: 100

Solution
--------

Count number of spaces and convert number to ASCII letter.

```python
with open("flag","r") as f:
    l = f.readlines()
    x = []
    for i in l:
        x.append(len(i) - 1)
    for i in x:
        print(chr(i))
```
